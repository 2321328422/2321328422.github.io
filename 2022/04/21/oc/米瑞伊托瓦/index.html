# 米瑞伊托瓦

# 设定

  • 名前：米瑞伊托瓦 (Miraitowa)
  • 性别：女
  • 诞生日：8.26
  • 年龄：222 (包含每次死后轮回的时间)
  • 种族：掌管轮回的神明 (成神前是人与伪神混血)
  • 能力：死后回到死前的时间线
    • 只有自杀才能摆脱轮回的痛苦
  • 武器：月耀 (魔法书)
    • 月耀包含了各种各样的魔杖
    • 同样也记录了各式咒语，还有米瑞伊托瓦自己写上去的

# 其他

  • 米瑞伊托瓦生活在一个被神明抛弃的世界，没有秩序鬼神横行，众伪神没有一位愿意站出来维护秩序
  • 因为流着伪神与人类的血液被称为杂种
  • 在她发现自己会重生的能力后她遇到一位白发的女子
    • 白发女子带着她生活了一段时间临走之前告诉她
    • Now|you're|coming|at|the|king|so|you|better|notmiss(想坐上王座那不容有失)
  • 月耀也是白发女子临走时留下的
  • 米瑞伊托瓦对谁都很温柔因此在路途中吃了不少亏
    不少种族对于米瑞伊托瓦想成为神明的行为进行嘲讽或者提供错误的消息
  • 她知道这样打听下去得到的信息的是没有价值的
    于是前往雪山去寻找传闻中不合群的古龙行星黎明
  • 最后以米瑞伊托瓦左眼为代价构建灵魂契约双方能力共享
  • 米瑞伊托瓦很喜欢行星黎明尽管只是合作关系但是她还是会下意识关心你
  • 一人一龙在世界的尽头击败了灾厄成为了神明
    米瑞伊托瓦喜欢花， 草， 书籍
    但是这些事物在那片大地上很难见到除了精灵族的领地
    成为神明后米瑞伊托瓦完成了自己的梦想在草地上舒服地睡一觉
  • 有传闻说这个世界被神明抛弃的原因是神明吞噬者在附近沉睡，也是众伪神不敢成神的原因
    但是行星黎明和米瑞伊托瓦不畏惧这个所谓的传闻
    直到她们看到巨大的蠕虫奄奄一息躺在世界的边界时
  • 神明对神明吞噬者的厌恶是本能
    但是米瑞伊托瓦不顾行星黎明的反对将变成小孩模样的神明吞噬者抱回家
  • 米瑞伊托瓦照顾了被未知诅咒无法维持原形的神明吞噬者一段时间并直呼对方为小吞 (行星黎明：只有你敢这么叫了)
  • 在一次处理伪神事件的时候察觉到了神明吞噬者对自己异样的占有欲
  • 一辈子没谈过恋爱的米瑞伊托瓦感受到了自己的情感，但是不敢面对
  • 在一次 C011@PS3 主动邀请她去散步时
    『我也能再次迎来春日吗』
    『美丽的花儿也会在你的内心盛开』

# 人际关系

  • 夫妻：C011@PS3
  • 挚友：行星黎明
  • 好友：立花染， 星川千， 决星呱
  • 恩师：Aikaid
