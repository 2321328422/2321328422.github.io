# 米瑞伊托瓦

# 设定

  • 名前：米瑞伊托瓦 (Miraitowa)
  • 性别：女
  • 诞生日：8.26
  • 身高：167cm
  • 年龄：222 (包含每次死后轮回的时间)
  • 种族：掌管轮回的神明 (成神前是人与伪神混血)
  • 能力：死后回到死前的时间线
    • 只有自杀才能摆脱轮回的痛苦
  • 武器：月耀 (魔法书)
    • 月耀包含了各种各样的魔杖
    • 同样也记录了各式咒语，还有米瑞伊托瓦自己写上去的

# 简介

  • 温柔，拥有悲惨的过去却依然在温暖他人并且总是自我鞭挞
  • 厨艺很好
  • 花， 草， 书籍​​​
  • 如何判断米瑞伊托瓦有没有再次轮回
    • 每次死后回到特定时间点米瑞伊托瓦的菱形瞳孔会转一圈
    • 还未成神明前米瑞伊托瓦的菱形瞳孔只会在轮回的出现
  • 米瑞伊托瓦脖子周围星环是在遇见神明吞噬者之后出现的
    • 好像是某人给她设的印记
    • 过了一段时间后米瑞伊托瓦会隐藏这个印记了
  • 米瑞伊托瓦和行星黎明在成神前经常会遇到各种各样的种族或者混血
    • 夜晚往往睡不了好觉，会遇到袭击或者一觉睡醒就回到之前时间线了根本不知道自己怎么死的.jpg
    • 一人一龙听到一点声音都会惊醒
    • 长期睡不好觉
    • 所以一人一龙的愿望都是睡个好觉
    • 成神之后能睡好觉了但是听到声音又会下意识惊醒
    • 米瑞伊托瓦是遇到神吞后有所好转
    • 在神吞成年后则是能够像正常人一样睡觉了
  • 米瑞伊托瓦喜欢阅读书籍
    • 家里有一个图书馆
    • 天气冷就在书房
    • 天气好就在草地的树下
  • 米瑞伊托瓦如果需要处理和思考的事物过多
    会看着看着书睡着大脑过载.jpg ​​​
  • 米瑞伊托瓦喜欢花
    • 有一片花园
    • 一开始并不怎么会养花
    • 然后慢慢学会了在行星黎明的帮助

# 人际关系

# 与行星黎明

  • 十分信任行星黎明，不言而喻的默契
  • 喜欢听行星黎明絮絮叨叨的说她的见闻

# 与 C011@PS3

  • 称呼对方为小吞
  • 深爱着对方
  • 因为爱着，所以米瑞伊托瓦愿意为对方付出一切

# 与星川千

  • 把星川千当女儿
  • 认为对方是个很可爱的孩子

# 经历

# 零

  • 米瑞伊托瓦生活在一个被神明抛弃的世界，没有秩序鬼神横行，众伪神没有一位愿意站出来维护秩序
  • 因为流着伪神与人类的血液被称为杂种
  • 在她发现自己会重生的能力后遇到一位白发的女子
    • 白发女子带着她生活了一段时间后临走之前告诉她
    • 想坐上王座那不容有失
  • 月耀也是白发女子临走时留下的
  • 米瑞伊托瓦对谁都很温柔因此在路途中吃了不少亏
    不少种族对于米瑞伊托瓦想成为神明的行为进行嘲讽或者提供错误的消息
  • 她知道这样打听下去得到的信息的是没有价值的
    于是前往雪山去寻找传闻中不合群的古龙行星黎明
  • 最后以米瑞伊托瓦左眼为代价构建灵魂契约双方能力共享
  • 米瑞伊托瓦很喜欢行星黎明尽管只是合作关系但是她还是会下意识关心你
  • 一人一龙在世界的尽头击败了灾厄成为了神明
    • 米瑞伊托瓦喜欢花， 草， 书籍
    • 但是这些事物在那片大地上很难见到除了精灵族的领地
    • 成为神明后米瑞伊托瓦完成了自己的梦想在草地上舒服地睡一觉
    • 瞎掉的左眼也因成神明后恢复视力
  • 没成神之前行星黎明和米瑞伊托瓦的信徒只有彼此
    • 那时的她们不需要其他的信徒也没有收过信徒
    • 米瑞伊托瓦成神明之后
    • 知晓她事迹的人自发组织宣传她的事迹，便有了不少信徒
    • 米瑞伊托瓦总是会去援助遇到困难的信徒

# 壹

  • 米瑞伊托瓦改造世界尽头之后
    • 那个荒芜的地方便有了四季
    • 只从书上见过的四季终于能真正感受到
    • 米瑞伊托瓦最喜欢的季节是春天
  • 有传闻说这个世界被神明抛弃的原因是神明吞噬者在附近沉睡，也是众伪神不敢成神的原因
    但是行星黎明和米瑞伊托瓦不畏惧这个所谓的传闻
    直到她们看到巨大的蠕虫奄奄一息躺在世界的边界时
  • 神明对神明吞噬者的厌恶是本能
    但是米瑞伊托瓦不顾行星黎明的反对将变成小孩模样的神明吞噬者抱回家
  • 米瑞伊托瓦是温柔的神明
    • 即便对方的神明吞噬者，她也会去照顾祂
    • 见到神吞醒后突然流泪，很是惊慌，误以为对方受到重伤
  • 米瑞伊托瓦劝神吞在此修养一阵
    • 对方也一口答应下来
  • 神明吞噬者在此修养的时间，米瑞伊托瓦发现对方竟意外地好相处
    • 不论什么事件祂都会帮助米瑞伊托瓦去处理
    • 米瑞伊托瓦也察觉到了自己与神吞之间奇异的羁绊
  • 直到一天，米瑞伊托瓦想起了一切
    • 祂的脖子周围出现了星环
    • 黄昏时节米瑞伊托瓦只喊上神明吞噬者一同散步
      『我也能再次迎来春日吗』
      『美丽的花儿也会在你的内心盛开』
