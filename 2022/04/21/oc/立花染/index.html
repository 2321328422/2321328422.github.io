# 立花染

# 设定

  • 名前：立花染
  • 性别：女
  • 诞生日：8.30
  • 年龄：36
  • 身高：173cm
  • 种族：人类与恶魔混血
  • 能力：没啥特别能力只是力气很大
    • 「蚀」
    • 很弱的魅魔的魅惑能力根本没什么用只有 Aikaid 自愿上钩
  • 牧场管理员之一
  • 以博大的温柔包容大家
  • 巨好的厨艺
  • 很会打游戏
  • 吃货
  • 绘画天赋

# 经历

  • 立花染从记事开始就在牧场生活
    接受成为管理员的教育
  • 牧场管理员每一个月可以提要求索求某一样物品
    立花染选择了书，游戏机与绘画材料
  • 她自己识字并为自己取了立花染这个名字
  • 正式成为管理员后很受孩子欢迎
    • 孩子们喜欢立花染立花染也喜欢孩子们
    • 牧场空间应该是隔绝的
      但是 Aikaid 却出现在了这里
      立花染对这个金色透明模样的孩子充满了好奇心
    • 不管怎样我想让这个孩子也加入我的再演
  • 在偷偷单独和 Aikaid 单独谈话说明这里情况后，对方明显表明出对恶魔很感兴趣
    • 双方定下协议，立花染用恶魔的情报作为交换，要求 Aikaid 以人类小孩的模样生活，一方面是为了不背恶魔发现，另一方面是安抚孩子们的情绪
    • 你的样子可以改变吗
    • 可以 Aikaid 透明的模样变成了实体
    • 但是我跟你说明了我的情况，你没有跟我说明你的情况，是不是有点不划算啊，不过，我可以等
  • 立花染为 Aikaid 取名为提拉星即使对方不是很喜欢这个名字，后来：真香 Aikaid：这是怎么把我家乡文字认成提拉星的呀
  • 立花染经常会编写有关森林的恐怖故事讲给孩子们听以防他们去森林
    • 她也告诫提拉星不要去森林那个地方
    • 因为每周她都会前往森林深处和检查者会面汇报本周情况以及领取物资
    • 如果这时有人闯进森林那么就会在森林里迷失方向
    • 去森林里找人是一件让立花染头痛的事情
  • 立花染跟往常一样去森林里面和检查者会面，却只见到物资没有见到检查者
    • 立花染心生疑虑
    • 有些事物正在改变
  • 立花染厨艺很好
    • 就算是因为协议模仿人类进食的提拉星也喜欢立绘染的饭菜
  • 会带提拉星玩当时特别流行的电子游戏
    • 提拉星经常玩不过立花染
    • 提拉星：再来一局，最后一局.jpg
  • 立花染会经常逗提拉星
    • 每当她去摸提拉星的头，对方就会红着脸把立花染的手甩开
    • 也会笑着捏她的脸，然后看着对方气鼓鼓的跑开
  • 两人在夜晚孩子们睡着后会去散步
    • 其实是交易的时间
    • 立花染一边讲着自己所知道的恶魔的情报，一边在纸上绘画着恶魔的面貌
  • 餐厅里面的儿童画都是立花染亲手画的
  • 提拉星会悄悄搞些高科技的东西
    • 在提拉星的帮助下，立花染通过屏幕看到了自己一直想知晓的，人类的世界
    • 原来人类是这样画画的，我也好想拥有他们那个 “板子” 啊
    • .... 很简单的东西，我造给你吧
    • 这算是礼物吗，我就收下啦立花染笑着看向她
    • 也，也算吧她涨红了脸
    • 在收到提拉星给立花染造的数位板后，立花染坚持要给她画肖像画
    • 画完后，她清楚地看到对方嘴角上扬了
  • 日子一天天过去就在立花染习惯了提拉星的存在后发现对方一直躲着自己
    • 跟往常一样的夜晚，但是聊完后立花染拉住了提拉星
    • 为什么要躲着我
    • 提拉星沉默不语
    • 在闹脾气吗，好像小孩子一样立花染笑着去摸对方的头
    • 不要把我当做小孩子这次对方恶狠狠的甩开了
    • 我从来没有真正把你当做小孩子立花染不再笑了
    • 那你这么做...
    • 因为我喜欢你立花染笑了，如灿烂的烈日
    • ... 我也提拉星低着头涨红了脸
  • 此后，两人的夜晚不再是为了交易
    • 提拉星跟立花染说起自己来到这里的原因，因为自己星球被未知的能量冲击毁灭，自己只能被迫降到这里
    • 立花染抱紧了提拉星
    • 如果我是上层恶魔派来的呢，或者我现在也在骗你呢
    • 我还是会喜欢你，谁叫是你呢
    • 提拉星也抱紧了立花染
  • 听了提拉星自己星球的故事后，立花染就开始一边画画一边骚扰提拉星
    • 星，你们星球科技这么发达一定有那种什么都能做到的机器手臂吧
    • 星，你看要是有机器手臂的话，配一个黑色外套更酷吧，像我的这件一样
    • 加上黑色短裙就更好了！立花染把设计草图摆给提拉星看
    • 被提拉星一口回绝了
  • 立花染取下自己的一边蓝色发带，绑在提拉星头发上面
    • 看对方没有拒绝，开心地笑了
  • 日子一天天过去，立花染感觉自己逐渐有点力不从心，她对提拉星说是自己有点累了
  • 但她知道是那个日子要到了
    • 她必须用蚀才能守住现状
    • 但在决定发动的那一刻
    • 立花染看到了提拉星脸上的表情
    • 她无法说出她心中的愿望
      最终从人脸堆砌的高台上坠下
  • 即便被提拉星救下立花染知道自己命不久矣
  • 「蚀」
    • 一些恶魔天生就拥有的能力
    • 与上古的魔之柱进行交易
    • 献祭你最重要的东西来实现愿望
    • 在感到最绝望的时刻会自动显现
  • 立花染
    • 在死亡之前发动了「蚀」
    • 献祭了自己最珍惜的孩子们换取这段时光的再来
    • 为了再看到他们的笑容就要再看见他们的死亡
  • 但是她看见了提拉星恐惧的表情
  • 对不起让你露出这种表情了
  • 永远睡了过去

  • 夫妻：Aikaid
  • 熊孩子：决星呱
