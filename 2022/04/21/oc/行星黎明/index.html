# 行星黎明

# 设定

  • 名前：行星黎明
  • 性别：女
  • 诞生日：2.17
  • 年龄：456 (包含死后轮回的时间)
  • 身高：175cm
  • 种族：古龙
  • 能力：治愈
    • 改变身体大小
    • 经过的地方会留下闪光，为米瑞伊托瓦提供暂时性的照明
    • 精通炼金魔法

# 简介

  • 丰富的知识储备
  • 厨艺很差
  • 话痨
    只有米瑞伊托瓦和星川千能够接受的程度
  • 米瑞伊托瓦和行星黎明在成神前经常会遇到各种各样的种族或者混血
    • 夜晚往往睡不了好觉，会遇到袭击或者一觉睡醒就回到之前时间线了根本不知道自己怎么死的.jpg
    • 一人一龙听到一点声音都会惊醒
    • 长期睡不好觉
    • 所以一人一龙的愿望都是睡个好觉
    • 成神之后能睡好觉了但是听到声音又会下意识惊醒
    • 行星黎明是遇到星川千之后有所好转
    • 搂着星川千睡觉让她觉得很安心
  • 最喜欢的花是满天星

# 人际关系

# 与米瑞伊托瓦

  • 十分信任米瑞伊托瓦，不言而喻的默契
  • 喜欢吃米瑞伊托瓦做的饭菜
    • 从此胃口被养挑
  • 喜欢什么事情都跟对方分享
    • 对方总是静默听着从不打断她

# 与星川千

  • 对星川千的称呼：小千，千
  • 龙对自己的财宝都有很强烈的占有欲
    • 行星黎明对知识很看重
    • 但是遇到星川千后她认为对方就是最重要的财宝
  • 星川千的学历只有初中
    行星黎明有时会给星川千讲课看着星川千睡着后会抱着她回卧室
  • 洗完澡星川千会窝在行星黎明怀里让她用风魔法弄干
  • 行星黎明一般用水魔法清理身体
    • 星川千不喜欢用水魔法清理身体
    • 所以两人会在龙巢里面巨大的水池中洗澡
    • 行星黎明会调节水温，用龙形态戏水单纯因为和星川千一起洗澡很开心
    • 也会用人形帮她搓澡浴池 play（？）
  • 关于两个人住的地方
    • 大部分在龙巢
    • 有时候星川千想回去住
    • 行星黎明也会陪她去 ​​​
  • 为什么会喜欢星川千
    做手工认真的眼神，我带着她飞时发自内心的笑容和那闪闪发光的眼睛，是最珍奇最吸引我的宝石

# 与 C011@PS3

  • 看不爽但是不敢打
  • 对 C011@PS3 的称呼：那条虫
  • 后面渐渐接受了对方的存在

# 经历

  • 古龙一族追求知识，行星黎明也不例外
  • 行星黎明并不合群，她认为古龙族的知识被拘束在这个世界里面，她想了解更多
  • 米瑞伊托瓦闯进自己的龙巢并对她说「请帮助我，我想成为神明」
    行星黎明早对她有所耳闻，等着她的到来有个神经病想要成为神明的消息已经传开了 然后就变成两个神经病要成为神明的消息传开了
  • 以米瑞伊托瓦左眼为代价构建灵魂契约双方能力共享
  • 在战斗中负责辅助米瑞伊托瓦
    也在旅途中给米瑞伊托瓦提供信息还有科普
  • 很惊讶米瑞伊托瓦为什么会拥有月耀这本魔法书
  • 行星黎明话很多唠叨但是米瑞伊托瓦不嫌烦
  • 多次的死亡轮回连身为古龙的行星黎明都受不了
    但是米瑞伊托瓦每次重生后第一件事就记录刚刚的错误再修改对策
    想坐上王座那不容有失 米瑞伊托瓦如是说
  • 行星黎明对米瑞伊托瓦的态度从那一刻开始转变
  • 一人一龙在世界的尽头击败了灾厄成为了神明
    米瑞伊托瓦在世界尽头造了草原和高山
  • 行星黎明将高山改造成自己的龙巢
    她如愿获得了这个世界以外的知识并且完成了自己的梦想用书堆满龙巢
  • 虽然有了自己的龙巢，但还是会和米瑞伊托瓦呆在一起
    行星黎明和米瑞伊托瓦已经到达十分默契的境界了
  • 行星黎明厨艺很差，经常来蹭饭
  • 米瑞伊托瓦不顾行星黎明反对将 C011@PS3 行星黎明：这种人都敢带回来你以为你真是圣母啊.jpg 带回来了之后便很少呆在米瑞伊托瓦身边
    神明对神明吞噬者的厌恶是本能
    行星黎明虽然嘴上受不了米瑞伊托瓦的圣母行为但还是会去援助米瑞伊托瓦
  • 在知晓米瑞伊托瓦和 C011@PS3 的事情之后
    行星黎明觉得这个家越来越没有我的容身之地了.jpg
  • 行星黎明动用神明的力量去往了别的世界遇到了差点嗝屁死亡的星川千
    在星川千世界的时候是人形状态，以往大部分的时候都是龙的形态
    行星黎明为了蹭口饭在星川千家赖着不走没想到星川千厨艺也很差
  • 行星黎明阅读过有关魔法少女的书籍
    由于第一次见真正的魔法少女，对星川千产生的浓厚的兴趣行星黎明：女人你引起了我的注意
  • 被星川千笨拙的战斗方式傻到，决定教导♀她
  • 行星黎明对掉落的龙鳞并不在意，但是星川千却收集起来做成了发卡行星黎明：dokidoki
    看到星川千脆弱的一面后决定救赎她
  • 与星川千表白并签订契约结婚证要求她成为自己的神使
  • 行星黎明头上的绿宝石装饰是星川千做的
    行星黎明并没有多少珠宝，星川千只好挑了绿宝石做材料行星黎明：早知道就多收藏珠宝了！！！
  • 行星黎明很喜欢炫耀自己有个老婆
    星川千：脸红
  • 行星黎明在之后接受了神明吞噬者的存在偶尔还是会吵架 米瑞伊托瓦：你们不要再打啦
