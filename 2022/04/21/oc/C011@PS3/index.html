# C011@PS3

# 设定

  • 名前：C011@PS3
  • 别名：虫崩 / 神明吞噬者 / 神吞
  • 性别：无
  • 诞生日：12.17
  • 年龄：未知
  • 身高：180cm
  • 种族：蠕虫
  • 能力：撕碎空间，吞噬

# 简介

  • 蠕虫家族的骄傲，众神畏惧的对象
    • C011@PS3 是蠕虫的语言意为，虫崩
    • 只有上层存在知晓他们的语言所以更多人称呼祂为神明吞噬者或神吞
  • 沉睡在宇宙某处的巨虫，以神明神灵为食
    • 感到饿时会出来觅食
    • 只会吞噬恶神
    • 每一次吞噬过后会沉睡很久
    • 讨厌被吵醒
  • 遇见米瑞伊托瓦后更加常用人形态
    • 本体是一条巨虫，身上布满了祂的眼睛
    • 不要与祂对视
  • 深爱着米瑞伊托瓦
  • 和星川千是手工同好
  • C011@PS3 喜欢睡觉
    • 讨厌睡觉时被吵醒
  • 濒死状态本体会变得很小
    • 转换成人体的话就是小孩的形态

# 人际关系

# 与米瑞伊托瓦

  • 对米瑞伊托瓦的称呼：瑞
  • 无论是现在还是过去都深爱着对方
  • 小孩形态时期会在米瑞伊托瓦看书的时候靠在她肩膀上睡觉
    • 长大后则是米瑞伊托瓦靠在神吞肩上睡觉​

# 与行星黎明

  • 对行星黎明的称呼：那条龙
  • 两人互相看不顺眼，看在米瑞伊托瓦的份上没有打起来
    • 在了解两人的经历后对行星黎明的态度有所好转
    • 不再是聊两句就要打起来的关系了

# 与星川千

  • 唯一一个被神明吞噬者认可的神使
    • 因为第一次见面星川千面部表情竟毫无波动
    • 一般的神使，神灵，神明对神吞都是恐惧，厌恶的态度
  • 为了给米瑞伊托瓦制作饰品向星川千请教手工的相关知识
    • 很佩服星川千作为一个如此年轻的神使竟不畏惧自己
    • 米瑞伊托瓦的眼镜就是 C011@PS3 制作的
    • 之后经常和星川千一起讨论手工

# 与玛栖霧

  • 十分厌恶玛栖霧

# 经历

# 零

  • 蠕虫家族的骄傲，虫崩，欲成为那不可能成为的人
    • 这一切的祂们眼中不过是虫豸妄想成龙罢了
  • 就算虫崩成为祂们的刃，成为一个刽子手
    • 也不得被封为神明
    • 仅仅是因为祂们对蠕虫的血脉感到恶心
  • 自出身起就被誉为最强的虫崩从未如此地绝望，它与祂们之间的差距如同深渊
    • 你将成为神明，成为吞噬神明的神明
    • 将六千层腐烂的神明，吞噬殆尽吧
    • 我叫，米瑞伊托瓦，掌管轮回的神明
    • 唯有她，唯有她米瑞伊托瓦为它回头
  • 瑞，我听说很多人都是在春日成神的，我也会迎来春日吗
  • 当然，美丽的花儿也会为你盛开
  • 在米瑞伊托瓦的帮助下，虫崩得以触及那神明的一角
    • 轮回的神明，若你不愿处决虫崩
    • 你将，失去纯正的血液，永陷轮回之苦
    • 得知米瑞伊托瓦暗地帮助虫崩的祂们介入其中
  • 我可是轮回的神明，小吞，就算你吃掉我，我也一定会再次和你相见的
    • 此刻，吞噬神明的神明诞生了
    • 名为神明吞噬者

# 壹

  • 经历了几千年的岁月，神明吞噬者仍坚守着与轮回神明的约定
    • 每一次吞噬后，米瑞伊托瓦的身影便再度浮现在祂的梦里
    • 就算吞噬了米瑞伊托瓦的魂，骨，肉，神吞还是将那一份记忆保留了下来
  • 直至一天被 26th 掌管空间的神明， 璃袭击
    • 至今最强的空间神明璃认为清理神明吞噬者是必要的
    • 尽管如此璃还是不敌神吞被祂吞噬
    • 神吞因此也受到重伤而后继续沉睡
  • 沉睡中的却被突然成为神明的行星黎明和米瑞伊托瓦吵醒
    • 醒来后发现寄生虫早已趁自己虚弱之时施展了 [诅咒]
    • 被寄生虫诅咒的神吞无法保持自身的完整形态陨落到了离自己最近的世界
  • 就这样死去也好
    • 或许就能见到你了吧
    • 但是，为什么这里会有龙？
  • 神吞快要失去意识之前被米瑞伊托瓦所救
  • 醒来后见到米瑞伊托瓦的神吞，第一次流泪了
    • 看到神吞流泪的米瑞伊托瓦慌慌张张地询问祂是不是哪里不舒服
    • 神吞更加确信，她就是祂的米瑞伊托瓦
    • 只不过没有之前的记忆
  • 神吞表示自己没事，并以修养为借口借住下来
    • 并借此机会再度接近米瑞伊托瓦
    • 对行星黎明的敌意毫不在意
  • 神吞会帮助刚成为神明的米瑞伊托瓦处理伪神事件
    • 就像当初米瑞伊托瓦教导神吞一样
  • 在和米瑞伊托瓦重新相处的过程中，神吞的力量也在恢复
    • 直到一天，神吞觉得自己保留的米瑞伊托瓦那一部分真在消散
    • 这让神吞十分惊恐，直到黄昏时节米瑞伊托瓦喊祂一同散步
  • 我也能再次迎来春日吗
  • 美... 美丽的花儿也会在你的内心盛开
