# 刻

# 设定

  • 名前：刻
  • 性别：无
  • 诞生日：4.21
  • 年龄：419
  • 身高：165cm
  • 种族：天使 (成为时间之神之前)
  • 能力：掌管时间的能力
    • 庞大的情绪能量

# 简介

  • 所属于长老会的天使
  • 喜欢吃抹茶布丁
  • 喜欢一个人呆在罂粟花田，因为安静
  • 自称自己是玻璃心
  • 刻是天使但是天生没有翅膀而受到歧视
    • 因为长期遭到冷暴力与霸凌性格担心自卑
  • 刻穿蒸汽朋克风服饰的原因是周围孤立她故意不给她发放制服
  • 刻害怕犯错
    • 一犯错就会被责备
    • 还有控制不住的发抖
  • 实际上就算不是刻的责任也会被指责
    • 第一个没有指责，反而纵容她的人是玛栖霧
  • 刻会在没人的地方偷偷哭

# 人际关系

# 与玛栖霧

  • 喜欢玛栖霧又想杀掉玛栖霧
  • 不知道是从什么时候习惯了玛栖霧的存在，不知道从什么时候想触碰他
    • 或许是从为祂带上罂粟花冠的那一瞬在刻心里埋下了种子，逐渐生根发芽
    • 为什么要不自量力，不惜自毁，明明知道这种攻击是无法伤害到你的。
      你大概猜到了吧，我确实是想拯救世界树，但在深埋在我内心那份情感....
      永远无法得到你的我，就这样死去吧

# 经历

  • 长老会内能够习得时间的天使越来少
    这届天使只有刻学会了操控时间的能力
  • 而刻的能力很弱
    长老会把希望放在下一届天使上， 忽略了刻的存在
  • 刻生性自卑胆小本以为终于能够成为一个有用的存在
    但长老会的忽视加重了刻对自己的否定
  • 刻为了更强，在情感派安的劝说下保留了情感
    • 情感派的天使代表安对谁都很温柔，它正因此出名
    • 越温柔越强大，这正是我们情感派的优势它这样对刻说
  • 正如安所说保留情感的刻变得更加强大了
    但是刻所用的是自身的负面情绪 —— 自卑
  • 刻将自己动用是负面情绪的事情隐瞒下来对外宣称自己的情绪能量来源是热情
  • 即便是动用了情感能量的刻也无法得到长老会的承认
    • 直到需要下一任掌管时间的神明的同时没有其他天使学会操控时间的能力时才想起了它
    • 在下一位操控时间的天使出现之前，就让它先担任吧
    • 担任这样重大的职责对天使而言是荣誉
    • 但是刻在神明仪式前一天才收到通知
  • 掌管时间与掌管空间的神明总是一起行动的
    但是身为无情感派 26th 掌管空间的神明璃拒绝和刻一起行动，它认为对方太弱了会拖后腿
  • 低估了对方力量的璃被神明吞噬者吞噬后
    • 长老会斥责刻不肯与璃一起行动的冷漠行为
    • 并在圣殿对刻进行圣水洗礼
    • 圣水洗礼对每一位天使来说是耻辱，只有有罪过的天使才需要圣水洗礼
  • 刻成为了第一位没有搭档的时间神
  • 在刻绝望之时，它遇到了影
    那位传说中的寄生虫母巢
  • 影自称自己对刻一见钟情
    • 情感只有世界树的产物才会拥有
    • 寄生虫是没有情感的，而影是寄生虫的母巢
  • 影一次又一次地从奇怪的空间里面冒出来找刻
    • 为了避免被看到的风险，每次影一出现它就会暂停时间
    • 影不受世界树法则的影响自然不会被暂停时间
  • 每一次见面都是影对刻进行各种言语的诱惑
    • 刻知晓了影不知从何处学到了操作空间的能力
    • 不受果的影响的祂更加肆意滥用
    • 刻明白了影接近自己是为了了解掌管空间的神明
  • 刻喜欢吃抹茶布丁，和安第一次见面的时候对方用抹茶布丁招待了刻
    • 一次看到影的时候刻脱口而出
    • 「好想吃Matchapudding(抹茶布丁)啊」
    • 原来亲爱的喜欢 Matchapudding 呀
    • 在下次的见面的时候亲爱的，以后就叫我玛栖霧吧，听说喊专属小名更能拉近我们之前的关系哦
    • 刻只是别过头没有作出回应
  • 刻有时候会呆在罂粟花田
    • 有一次刚好玛栖霧也跟了过来
    • 亲爱的这次怎么没有暂停时间呀，是想通了要公开我们的关系吗
    • 没有天使会来这种地方，它们认为罂粟花是丑陋的，而我认为罂粟花是美丽的
    • 刻做了一个罂粟花冠戴在了玛栖霧头上，放上去的那一刻罂粟花冠变成了荆棘
    • 不行吗....
    • 可以哦
    • 刻看见玛栖霧的样貌发生了改变，荆棘变回了罂粟花冠，刻下意识的笑了
    • 很美，和罂粟花一样美
  • 从来没有人见过玛栖霧的完整体，刻是第一个
    没有人这么在乎过我，第一个居然是寄生虫母巢，真讽刺
  • 之后刻好几天没有遇到玛栖霧
    甚至感到不习惯
    我居然在想祂，真是疯了
  • 直到一天玛栖霧拿着星杯出现在它的面前
    • 亲爱的，你想成为神吗，我是说，我们一起成为神
    • 这不可能... 从来没有两位神一起出现这一说，而且你是...
    • 因为我是世界树的背面我是影对吗，但是从来没有试过怎么知道呢，或许我们将创造历史
    • 玛栖霧用祂完美的空间切割术切开了星杯
  • 70th 与 71th 神同时诞生
  • 刻成为了神，第一个成为神的操控时间的神明
    • 同时所有人知道了它与寄生虫母巢不同寻常的关系
    • 长老会不再承认它是所属于长老会的天使
    • 对它异常的目光与鄙夷的声音越来越多
  • 亲爱的你都成为神了怎么还不开心，你现在明明可以把这些质疑你的苍蝇轻而易举的杀死
    .... 你不会懂的
  • 为了不让玛栖霧继续呆在神位上对世界树进行侵蚀
    • 刻偷偷参与了弑神兵器计划
    • 当实验基地的空间被玛栖霧摧毁时刻觉得最后的希望都被击破了
  • 在此之后刻第一次主动邀请玛栖霧见面
    亲爱的，终于打算接受我了吗，我好开心
  • 就在玛栖霧开玩笑的时刻，她低下头亲吻了祂的嘴唇
    并且动用了自己一直以来积累的庞大的负面能量
    我爱你玛栖霧，但是我必须杀了你
  • 70th 与 71th 神同时死亡
