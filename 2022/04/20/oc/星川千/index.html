# 古龙神的神使

# 设定

  • 名前：星川千
  • 性别：女
  • 诞生日：2.27
  • 年龄：18
  • 身高：158cm
  • 种族：魔法少女 / 神使
  • 能力：瞬间移动
  • 武器：刺剑

# 简介

  • 天生患有失语症
    • 因不会表达情绪所以内心活动十分丰富
    • 成为神使后失语症渐渐好转
  • 向往自由的少女
  • 喜欢蓝天，喜欢鸟
  • 擅长手工
  • 厨艺比行星黎明更差
  • 在天气好的时候会在龙巢附近放风筝
  • 最喜欢的糖果是影子糖不是喜欢吃是喜欢外观
  • 最喜欢的甜品是抹茶水晶冻
  • 最喜欢的饮品是蓝柑苏打水
    • 对星川千来说很贵虽然喜欢但是很少喝
    • 和行星黎明同居后行星黎明会制作蓝柑苏打水给星川千
  • 最喜欢的花是坠羽蓝星花
    • 星川千第一次遇见这个花是在米瑞伊托瓦的花园里面
    • 米瑞伊托瓦知道星川千喜欢坠羽蓝星花后赠了她几盆
    • 行星黎明知道星川千喜欢坠羽蓝星花后在龙巢附近种植了坠羽蓝星花田

# 人际关系

# 与行星黎明

  • 对行星黎明的称呼：黎明，蠢龙
  • 星川千不喜欢用水魔法清理身体
    • 所以会在龙巢里面巨大的水池中洗澡
    • 行星黎明会调节水温，用龙形态戏水单纯因为和星川千一起洗澡很开心
    • 也会用人形帮她搓澡浴池 play（？）
  • 星川千的学历只有初中
    行星黎明有时会给星川千讲课然后睡着
  • 洗完澡星川千会窝在行星黎明怀里让她用风魔法弄干
  • 关于两个人住的地方
    • 大部分在龙巢
    • 有时候星川千想回去住
    • 行星黎明也会陪她去
  • 为什么会喜欢行星黎明
    因为她对我伸出了手
R18相关

行星黎明虽然是雌龙但是成神之后没有性别
（上半身是雌性下半身是雄性）
行星黎明在床上对星川千比较粗鲁（急躁）
因为行星黎明极强的占有欲想要证明星川千是属于祂的
听到星川千喊痛又会心疼
为了星川千能够承受自己
有时会在星川千喊痛时给她喝自己的龙血
有时也会在床事之前给星川千喝一些魔法药
事后行星黎明会用水魔法清理星川千的身子然后抱着她入睡
星川千觉得每次起来腰都散架
回忆起火热的夜晚会不禁脸红
回忆起和行星黎明交融时听到祂在自己耳边低声说 “我爱你” 时想找个洞钻进去
行星黎明喜欢睡醒后看着星川千脸红的样子揉她的头
“蠢龙… 快停下，我不行了…”
“没关系，千，我会给你龙血的” ​​

# 与米瑞伊托瓦

  • 称米瑞伊托瓦为米妈妈
  • 喜欢米瑞伊托瓦，像妈妈一样待人温柔

# 与 C011@PS3

  • C011@PS3 为了给米瑞伊托瓦制作礼物向星川千请教手工的相关知识
    • C011@PS3 天天看行星黎明炫耀决定自己也要给米瑞伊托瓦做一个
    • 其实一开始很害怕 C011@PS3 但是没有表现出来内心慌得一批
    • 之后熟了敢直呼 C011@PS3 为神吞星川千：C011@PS3 这什么名字太绕口了
  • 从此和 C011@PS3 是手工同好

# 与决星呱

  • 在魔女战斗的时候与中途插进来的决星呱有一面之缘

# 经历

  • 星川千自幼因失语症遭到异样的眼光
  • 想逃离被囚禁的生活与 qb 签下契约的魔法少女
    • 愿望是让我离开这里去哪里都好
    • 生父死后被亲戚收养
  • 大部分时间一个人生活
  • 因为长期与外界隔绝而面部表情少少言寡语三无性格
  • 在一次战斗中差点丧命，被行星黎明所救
  • 自此之后开始同居并教星川千战斗技巧
  • 知晓魔法少女真相后崩溃被行星黎明救回
    星川千总是一副表面什么都不在意的样子，但是知晓真相的她还是露出脆弱的一面
  • 头上的发卡是用行星黎明的龙鳞做成
  • 与行星黎明签订契约，成为古龙神的神使
    之后在龙巢生活
    应该是最弱最年轻的神使了
  • 一个在圆环之理出现之前的时间线上被救赎的魔法少女
  • 为了报答行星黎明给她制作了绿珠宝为材料的饰品
    行星黎明身上大部分饰品都是星川千制作的老婆做的就是好看
