# 世界观

# 设定

  • 从世界诞生的那一刻，就出现了光，有了光就会有影
  • 所有的​物种都诞生于世界树
  • 除了寄生虫

# 神明

  • 神明 掌管世界某方面的神 如掌管时间的神明 掌管空间的神明
  • 成为神明需要神灵或者神明或者神的指引
  • 不会因为时间的流逝死去，但会被杀死

# 神灵

  • 通过他人信仰而诞生
  • 信仰决定他们的能力，有些神灵可通过庞大的信仰成为神明
  • 靠信仰存活，只要信仰存在，被杀死也能够再生
  • 如果信仰消逝就会死亡

# 神

  • 神明与神灵有很多但是神只有一位
  • 神通过「星杯」诞生

# 星杯

  • 「星杯」​与生命树一起诞生的产物获得它 你既是神
  • 既能获得 也能被夺去
  • 第一位神被『朗基奴斯枪』杀死后「星杯」与神的诞生离不开战争与牺牲
  • 但随着时间的推移与神的更迭「星杯」的下落越来越模糊不清

# 寄生虫

  • 所有的寄生虫都来自于他们的母巢 影
  • 它们对母巢绝对服从
  • 它们不受世界树法则的影响
  • 它们蚕食世界树的产物来生存

# 伪神

  • 次于神灵与神明的存在
  • 也有可以成为神灵或神明却自愿呆在伪神这个位置的伪神

# 世界

  • 不同世界的存在不同的秩序
  • 不同的种族生活在不同的世界
    • 也有混合种族的世界米瑞伊托瓦的世界
    • 也有被其他种族入侵的世界
  • 通常每个世界的神明会主动维护所在世界的秩序
  • 也有放任不管的存在

# 果

  • 时间与空间这样强大的力量带来的副作用很大
  • 掌管时间与掌管空间的神明在使用能力的时候会积攒果
  • 长老会通过向世界树祈祷来消除果
  • 这也是他人不会滥用这样强大力量的原因

# 地狱

  • 分为下界和魔界两个区域
  • 适宜魔族生存的地方
  • 等级：低中高阶魔物 < 低中高阶魔兽 < 低中高阶恶魔 < 魔王
    • 魔物，魔兽，恶魔可以有很多，但是魔王只能有一位
    • 魔王可以是任何种族只有你有足够的实力杀死在任魔王
    • 无论多么卑劣龌龊的手段都可以使用，这里可是地狱
  • 等级不意味地位，但意味实力
  • 拥有实力并不能代表你能够在地狱过得很好
  • 所以地位也是很重要的
  • 高层的恶魔总是会通过各种手段把资源聚集在他们手里

# 魔界

  • 由三十三座城池组成，除去魔王城以外，其他三十二城池的城主是按实力排名的
  • 由于每个城主的处理事物的方法不同，所以不同城池之间的环境和秩序也不同
  • 当上城主的方法和魔王同样，击败在任城主

# 下界

  • 被称为真正的地狱
  • 只有岩浆和尸骨
  • 传闻是一切不可言不可直视诡异邪恶之物的栖息地
  • 传闻「蚀」是源于一个从下界回来的恶魔
  • 「蚀」
    • 一些恶魔天生就拥有的能力
    • 与上古的魔之柱进行交易
    • 献祭你最重要的东西来实现愿望
    • 在感到最绝望的时刻会自动显现
    • 你能够实现的愿望取决于你对要献祭的物有多珍惜
    • 魔之柱无法实现超出它自身能力范围的愿望，比如逆转时间

# 牧场

  • 高层恶魔为了吃到人类孩子刚成年时的灵魂而设定的机构
  • 通常要求人魔混血来当管理员
    • 人魔混血不受人类欢迎也不受恶魔欢迎，走投无路的人魔混血往往会来当管理员
    • 人魔混血寿命很短，但是从外表上看看不出年龄早上还好好的人晚上就嗝屁了
  • 高层恶魔在与管理员签订契约时削减了人魔混血的寿命在孩子们长大成年的那一刻，也是管理员的死期，绝望中的人类对他们来说更加美味
  • 这个秘密被高层隐瞒下来，让别人相信只是人魔混血命短而已

# 天堂

  • 被称为离世界树最近的地方
  • 天使没有性别，是由世界树直接孕育出来的种族
  • 等级：下阶天使 < 中阶天使 < 上阶天使 < 长老 < 大长老
  • 等级不是由自身实力而是由你对世界树的虔诚程度决定，而虔诚程度取决与长老们的判断
  • 大长老是在天堂中最高的位置
    • 需要获得在任大长老和长老们的认可
    • 或者做出巨大贡献超过在任大长老
    • 或者得到在任神的认可

# 长老会

  • 由天使自发组织成立的一个辅助神的机构
  • 是由长老们负责的组织所以被称为长老会
  • 分为无感情派和感情派
  • 负责帮助灵魂转生
  • 负责培养重要的掌管时间与掌管空间的神明其实是被长老会垄断了
  • 担任为每任掌管时间与掌管空间的神明消除果的责任
  • 随着人员的更迭，长老会从辅助神慢慢向干涉神转变

# 天使

  • 除去常规等级的天使，还有等级体制外的天使
  • 因为职位特殊和能力特殊原因没有等级
  • 精灵天使
    • 精灵天使是没有感情的
    • 负责收集做出贡献巨大的人的灵魂
    • 负责守护天堂
  • 祝福天使
    • 只有个位数存在的天使
    • 具有出口成真的能力
    • 每次发动消耗寿命

# 种族

# 德曼普勒星人

  • 是世界树孕育的生命中科技文明发展最高的种族
  • 他们崇尚科学追求科技的发展
  • 在漫长岁月中不断进化的他们脱去了肉体，变成了能量体
  • 从而更能将全身心投入到科研当中 ​​​
